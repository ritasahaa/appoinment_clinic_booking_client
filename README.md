# Appointment Clinic Booking Admin - Frontend

React.js frontend for the Appointment Clinic Booking Admin system.

## ⚠️ Important Note

It is designed for interview/training purposes to test a developer's ability to integrate REST APIs.

## Features

- Modern React with Vite
- React Router for navigation
- Axios for API calls (integration incomplete)
- Responsive design
- Help drawer on all pages

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

The frontend will run on `http://localhost:5173`

## Pages

- Login / Signup / Forgot Password / Reset Password
- Dashboard with statistics
- Doctor List and Details
- Patient List and Details
- Appointment List and Details
